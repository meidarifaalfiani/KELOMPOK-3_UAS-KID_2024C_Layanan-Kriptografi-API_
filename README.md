# KELOMPOK-3_UAS-KID_2024C_Layanan-Kriptografi-API_
Repositori ini dibangun menggunakan FastAPI dan dilengkapi fitur kriptografi dasar untuk menjaga integritas dan keamanan pertukaran data antar peneliti.  Project ini dibuat sebagai bagian dari tugas UAS Kriptografi &amp; Keamanan Informasi, dengan fokus pada penerapan konsep public key, digital signature, dan secure session menggunakan JWT.
